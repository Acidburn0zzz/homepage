Try it now

You can try Manjaro without changing your running system by putting it on a USB stick or on a DVD. If you decide to install Manjaro, you can do it directly from this live-system.

  • Try Manjaro in a virtual machine within your current operating system
  • Try Manjaro by putting it onto a DVD or USB-Stick
  • Install Manjaro from a DVD or USB-Stick

For simplicity we have selected XFCE and KDE for our flagship editions. If you want to look for other experiences, our community has created many flavours that focus around other desktops.
