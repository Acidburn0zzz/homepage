Linux fast and stable.

The easy to install Linux distribution with a friendly feel that comes with the latest and greatest apps at your fingertips.

Downloads & instructions

Grows with you

Built a powerful Arch-Linux foundation, with speed, efficiency and cutting edge software, Manjaro makes it stable and takes full advantage of your computer hardware. It is designed to make beginners happy and grow with them while they are becoming advanced users. It is easy to install even for beginners and comes with all the apps you need every day.


Solid base for everyone

Proven stability with newest features

You get the newest software - carefully tested by the Manjaro team and the community.

Easy to set up

Set up your hardware and preferences just by a few clicks.

Handpicked software

Manjaro comes with a compilation of handpicked software that has been proven to be useful, modern and reliable.

Choose your workflow

We support as many different environments as possible, so you can select the best for yourself.

Features that will help you

Manjaro comes preinstalled with GIMP, which is a very powerful image manipulation program. For drawing Krita will fulfill most of your needs, while Darktable brings a top-notch and free photography suite. For simple or advanced video-editing, Kdenlive and OpenShot will do almost anything you need. There are also professional video-editing tools like Lightworks. 3D-Modelling can be done using Blender. Most of this software is completely free available and can be easily and secure installed by using Manjaro’s integrated package manager.

Features of this edition

  • Comes with a tiling window manager that ensures maximum efficiency
  • Very resource-efficient and therefore ideal for older computers
  • Good for very advanced users

Be secure

We encourage you to check if the download was not modified by some malicious person or by an error while downloading.
