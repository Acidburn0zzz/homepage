What is Manjaro?

Manjaro is a user-friendly and open-source Linux distribution. It provides all the benefits of cutting edge software combined with a focus on user-friendliness and accessibility, making it suitable for newcomers as well as experienced Linux users.


What Manjaro will offer to you

Latest features with stability

You get the latest software, carefully tested by our Manjaro Team & Community before it reaches you.

Easy to setup

Our Hardware detection and preferences tool detects, installs on the fly without need for user to define and install.

Software for everyone

Manjaro is loaded with software of every category to suit needs of everyone. All you need is to choose, install and start using.

Choose your workflow

Manjaro comes in many flavors; You have the option to pick the desktop environment / window managers of your choice.
